# 💧 Water System
**ระบบบริหารจัดการน้ำประปาอัจฉริยะ** ที่เชื่อมโยงผู้ใช้ 3 กลุ่ม (Admin, Officer, Customer) เข้าด้วยกันผ่านฐานข้อมูลเดียว เพื่อการบริหารจัดการน้ำที่มีประสิทธิภาพและโปร่งใส

---

## 📖 สรุปภาพรวมระบบ (System Overview)
ระบบถูกออกแบบมาเพื่อรองรับการใช้งานที่ครอบคลุมทุกมิติของการประปา โดยแบ่งสิทธิ์การใช้งานตามบทบาท (Role-based Access Control):

* **👨‍💼 Admin (ผู้ดูแล):** จัดการบิลค่าน้ำ, ข้อมูลลูกค้า/พนักงาน และมอบหมายงานซ่อม
* **👷 Officer (เจ้าหน้าที่):** ติดตามคุณภาพน้ำ (ความขุ่น/คลอรีน), รับงานซ่อมตามโซน และดูแผนที่พื้นที่รับผิดชอบ
* **👤 Customer (ลูกค้า):** ดูบิลค่าน้ำ, จ่ายเงินผ่าน QR Code และแจ้งซ่อมด้วยการปักหมุดพิกัด

---

## ✨ การออกแบบและโทนสี (Brand Color Palette)
เน้นความสะอาด น่าเชื่อถือ และอ่านข้อมูลได้ง่าย (Data Visualization Optimized):

| ประเภท | สี | ความหมาย |
| :--- | :--- | :--- |
| **Primary** | 🟦 Sky Blue | สีหลักของแบรนด์ สื่อถึง "น้ำ" และ "การบริการ" |
| **Success** | 🟩 Green | ใช้กับปุ่มออกบิล, รับงาน และสถานะน้ำดี (ปลอดภัย) |
| **Alert** | 🟧 Orange/Red | การแจ้งเตือน, ยอดค้างชำระ และค่าคุณภาพน้ำที่ผิดปกติ |
| **Background** | ⬜ Clean Gray | พื้นหลังที่ช่วยให้ตารางและเกจวัดค่าต่างๆ โดดเด่น |

## 🛠 จุดเด่นของระบบ (Key Features)

### 1. แดชบอร์ดอัจฉริยะ (Smart Dashboard)
* **KPI Tracking:** แสดงปริมาณน้ำคงเหลือ, สถานะระบบ และคุณภาพน้ำ NTU แบบสดๆ
* **Water Level Visualizer:** กราฟระดับน้ำแนวตั้ง ช่วยให้เห็นปริมาณน้ำในถังเก็บได้ทันทีโดยไม่ต้องอ่านตัวเลข
* **Quick Access:** เมนูทางลัดขนาดใหญ่ แยกตามสิทธิ์การใช้งาน เข้าถึงฟังก์ชันหลักได้ในคลิกเดียว

### 2. ระบบจัดการบิลและชำระเงิน (Billing & E-Payment)
* **Meter Search:** ค้นหารหัสจากมิเตอร์เพื่อดึงข้อมูลสมาชิกได้ทันที
* **Digital Invoicing:** คำนวณค่าน้ำอัตโนมัติและส่งบิลออนไลน์ไปยังหน้าจอของลูกค้า
* **Secure Payment:** รองรับการจ่ายผ่าน QR Code พร้อมระบบอัปโหลดสลิปเพื่อยืนยันยอด

### 3. บริหารงานซ่อมและโซนพื้นที่ (Maintenance & Zoning)
* **Task Management:** ระบบจัดการงานซ่อม (Pending / In Progress / Done) เจ้าหน้าที่กดรับงานได้จากหน้าเว็บ
* **Geospatial Mapping:** แสดงแผนที่โซนรับผิดชอบ (Z01-Z04) พร้อมพิกัดที่ลูกค้าปักหมุดแจ้งซ่อมอย่างแม่นยำ
* **Repair Reporting:** ระบบแจ้งซ่อมระบุประเภทปัญหา (ท่อแตก, น้ำไม่ไหล) พร้อม Notification ถึงแอดมิน

### 4. วิเคราะห์คุณภาพน้ำ (Water Quality Report)
* **Analytical Cards:** แยกแสดงค่าความขุ่น (Turbidity) และคลอรีน (Chlorine)
* **Standard Reference:** มีตารางเทียบเกณฑ์มาตรฐานจากกรมอนามัยในตัว
* **Status Indicators:** ใช้สี (เขียว/ส้ม/แดง) เพื่อระบุความปลอดภัยในการอุปโภคบริโภค

## 💻 เทคนิคัลสแต็ก (Technical Stack)
* **Engine:** Node-RED (ทำเว็บ, จำลองค่า)
* **Database:** MariaDB (ฐานข้อมูล)
* **Frontend:** HTML5, CSS3, JavaScript (เขียนเว็บและตกแต่ง)

## 🚀 ขั้นตอนการติดตั้ง (Installation Guide)

โปรเจกต์นี้รองรับการรันผ่าน **Docker** เพื่อความสะดวกรวดเร็วในการตั้งค่า Environment ทั้ง Node-RED และ MariaDB

###สิ่งที่ต้องเตรียม
* ติดตั้ง [Docker Desktop](https://www.docker.com/products/docker-desktop/) ลงในเครื่อง

### ขั้นตอนการ Setup
1. **เตรียมไฟล์:**
   * ทำการดาวน์โหลดไฟล์ `iot-stack` และแตกไฟล์ (Extract) ลงในโฟลเดอร์ที่ต้องการ

2. **เปิด Terminal / Command Prompt:**
   * ใช้คำสั่ง `cd` เพื่อเข้าไปยังโฟลเดอร์ที่แตกไฟล์ไว้ (iot-stack)
     cd iot-stack
   
3. **รันระบบด้วย Docker Compose
4. *docker-compose up -d
