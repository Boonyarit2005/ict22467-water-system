# 💧 Water System
**ระบบบริหารจัดการน้ำประปาอัจฉริยะ (All-in-One Dashboard)**

---

## 📋 ภาพรวมระบบ (System Overview)
ระบบบริหารจัดการน้ำประปาที่เชื่อมโยงผู้ใช้ 3 กลุ่มเข้าด้วยกันผ่านฐานข้อมูลเดียว เพื่อเพิ่มประสิทธิภาพการทำงานและการบริการที่โปร่งใส:

* **👨‍💼 Admin (ผู้ดูแล):** จัดการบิลค่าน้ำ, ข้อมูลลูกค้า/พนักงาน และมอบหมายงานซ่อม
* **👷 Officer (เจ้าหน้าที่):** ติดตามคุณภาพน้ำ (ความขุ่น/คลอรีน), รับงานซ่อมตามโซน และดูแผนที่พื้นที่รับผิดชอบ
* **👤 Customer (ลูกค้า):** ดูบิลค่าน้ำ, จ่ายเงินผ่าน QR Code และแจ้งซ่อมด้วยการปักหมุดพิกัด

---

## ✨ การออกแบบและโทนสี (Brand Identity)
ยึดหลักการออกแบบที่สะอาดตา (Clean UI) เพื่อการอ่านข้อมูลที่แม่นยำ:
* 🟦 **Sky Blue (สีหลัก):** ใช้ใน Header และปุ่มยืนยัน สื่อถึงน้ำและการบริการ
* 🟩 **Success Green (สีดำเนินการ):** ใช้กับปุ่มออกบิล, รับงาน และสถานะคุณภาพน้ำปกติ
* 🟧 **Alert (สีเน้น):** ใช้กับการแจ้งเตือน, ยอดค้างชำระ และค่าความขุ่นน้ำที่เกินมาตรฐาน
* ⬜ **Clean Gray (พื้นหลัง):** ช่วยให้ข้อมูลในตารางและเกจวัด (Gauge) อ่านง่าย

## 🛠 จุดเด่นของระบบ (Key Features)

### 1. แดชบอร์ดอัจฉริยะ (Smart Dashboard)
* **KPI Tracking:** แสดงปริมาณน้ำคงเหลือ, คุณภาพน้ำ (NTU) และจำนวนการแจ้งเตือนแบบ Real-time
* **Water Level Visualizer:** กราฟระดับน้ำแนวตั้ง เห็นปริมาณน้ำในถังเก็บได้ทันทีโดยไม่ต้องตีความตัวเลข
* **Quick Access Menus:** เมนูทางลัดขนาดใหญ่แยกตามสิทธิ์การใช้งาน เข้าถึงฟังก์ชันหลักในคลิกเดียว

### 2. ระบบจัดการบิลและชำระเงิน (Billing & E-Payment)
* **Meter Search:** ค้นหารหัสจากมิเตอร์เพื่อดึงข้อมูลสมาชิกและคำนวณค่าน้ำอัตโนมัติ
* **Digital Invoicing:** ออกบิลออนไลน์และส่งข้อมูลไปยังหน้าจอของลูกค้าทันที
* **Secure Payment:** ลูกค้าชำระเงินผ่าน QR Code พร้อมระบบอัปโหลดสลิปยืนยันความโปร่งใส

### 3. บริหารงานซ่อมและโซนพื้นที่ (Maintenance & Zoning)
* **Task Management:** ระบบสถานะงาน (Pending / In Progress / Done) เจ้าหน้าที่กดรับงานได้ทันที
* **Geospatial Mapping:** แสดงแผนที่โซน (Z01-Z04) และพิกัดแจ้งซ่อมที่ลูกค้าปักหมุดไว้
* **Repair Reporting:** แจ้งปัญหาท่อแตก/น้ำไม่ไหล พร้อมส่ง Notification ถึงแอดมินทันที

### 4. วิเคราะห์คุณภาพน้ำ (Water Quality Report)
* **Analytical Cards:** แสดงค่าความขุ่น (Turbidity) และคลอรีน (Chlorine)
* **Standard Reference:** ตารางเทียบเกณฑ์มาตรฐานจากกรมอนามัยเพื่อความปลอดภัยของผู้บริโภค

## 🚀 ขั้นตอนการติดตั้ง (Installation)

ระบบนี้รันบน **Docker** เพื่อความง่ายในการจัดการ Environment:

1.  **เตรียมไฟล์:** ดาวน์โหลดและแตกไฟล์ `iot-stack` ลงในโฟลเดอร์ที่ต้องการ
3.  **เปิด Command Prompt / Terminal:** เข้าไปยังโฟลเดอร์ที่แตกไฟล์ไว้
4.  ใช้คำสั่ง cd โฟลเดอร์ iot-stack
5.  **สั่งรันระบบ:** ใช้คำสั่ง Docker Compose เพื่อสร้าง Container
    ```bash
    docker compose up -d
    ```
6.  **เข้าใช้งาน:**
    * **Dashboard:** `http://localhost:1880/login`
    * **Node-RED Editor:** `http://localhost:1880`
    * **Database:** MariaDB `http://localhost:8080`
    * MariaDB user root password 1234
